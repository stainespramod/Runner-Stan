# Runner Stan 🎮
A 2D chase game built using GameMaker Studio where the player must navigate a series of mazes while being chased by an enemy.

## ▶️ How to Play
- Use WASD keys to move
- Navigate your way through the maze while avoiding the enemy
- Reach the end to move on to the next level
